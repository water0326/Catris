## How to play

### Control

Move : W, A, S, D<br/>
Drop : Space<br/>
Reset Block : R<br/>

### Description

It seems like we are playing acommon Tetris game, but it might be different!<br/>
You can create blocks by controlling your cats which come in various lengths.
